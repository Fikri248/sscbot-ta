Daftar Tautan dan Link Penting Layanan SSC (Student Service Center) Telkom University:

1. Link template LPJ (Laporan Pertanggungjawaban):
Mahasiswa dapat mengunduh template LPJ melalui tautan berikut: https://linktree.ssc/lpj

2. Link proposal kegiatan:
Untuk mengajukan kegiatan, silakan gunakan template proposal kegiatan di tautan berikut: https://linktree.ssc/proposal

3. Link panduan TAK (Transkrip Aktivitas Kemahasiswaan):
Aturan, syarat, dan prosedur lengkap mengenai TAK dapat dilihat pada panduan TAK di tautan berikut: https://linktree.ssc/tak

4. Link dari Linktree SSC:
Untuk mengakses semua layanan, formulir, dan informasi terbaru lainnya, mahasiswa bisa mengunjungi Linktree resmi SSC di: https://linktr.ee/ssctelu

Gunakan tautan-tautan di atas untuk keperluan administrasi kegiatan kemahasiswaan Anda.
